## Project Overview

The Movie Page project is a **responsive web application** that fetches movie data from an API and displays it in a clean, user-friendly interface. Users can explore popular movies, view movie details such as ratings, release dates, genres, and watch trailers.

---

## Features

- Browse popular movies dynamically.  
- View detailed information for each movie (title, rating, release date, overview).  
- Play movie trailers directly on the page.  
- Responsive design suitable for desktop, tablet, and mobile.  
- Interactive UI with smooth animations.
